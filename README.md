# AI Resume Builder

An AI-powered Resume and Cover Letter Generator built with React.js, Vite, and Groq API.

## Features

- Generate AI-powered resumes
- Generate tailored cover letters
- Download cover letters
- Fast and responsive UI
- Groq API integration

## Tech Stack

- React.js
- Vite
- JavaScript
- Groq API
